# The Hair Witch — Website Option A (Monochrome)

**Preview site for Theresa Coggiola / The Hair Witch (Asheville, NC).**
Date: 2026-07-04

This is one of **three** design options built for review. Theresa can view each
one separately and choose the one she likes best — or a blend of the three.

- **Option A — Monochrome** (this repo): stark black/charcoal + gold, all-caps,
  gallery-forward.
- Option B — Editorial: warmer, magazine-style layout.
- Option C — Replica: a faithful mirror of her current live site.

## View it
